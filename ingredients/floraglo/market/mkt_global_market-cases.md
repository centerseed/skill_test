## 配方設計觀察

### 最常見基礎配方（低差異化）

```
FloraGLO® 葉黃素 20mg + 玉米黃素 1–4mg
```

約 40% 的歐美產品停留在此配方，依靠 FloraGLO® 品牌本身的臨床研究背書作為核心賣點。

### 差異化配方方向

**AREDS2 對標型**：FloraGLO® + ZeaONE® 按 5:1 或 10:2 比例  
- 代表：大研生醫視易適、露奇亞 Ruijia  
- 訴求：明確對應臨床研究數字，科學感強

**眼腦協同型**：FloraGLO® + Omega-3 / Neumentix®  
- 代表：Nordic Naturals Vision Support、BioJOY 愛維視  
- 訴求：葉黃素在腦部的積累（佔類胡蘿蔔素 66–77%），認知健康延伸訴求

**全效護眼型（台灣特色）**：FloraGLO® + ZeaONE® + 蝦紅素 + 花青素 + 玻尿酸  
- 代表：YohoPower 12合1、聚焦葉黃素 8合1、赫而司開明 8合1  
- 訴求：一粒全覆蓋，視覺系統多維防護，台灣市場溢價空間最大

**高劑量 / 技術差異型**：4x 超萃技術、30mg 高劑量  
- 代表：NICHIE（JP）、AFC Japan US  
- 訴求：劑量對標研究機構上限，吸收技術強調

**機能性食品精準型（日本特色）**：葉黃素 10mg ± 蝦紅素 6mg，SR 認可聲稱  
- 代表：Fancl えんきん、小林製薬 ルテインf  
- 訴求：受消費者廳認可的精確機能聲稱（黃斑色素密度、對比敏感度）

## 市場空白與機會

| 觀察 | 說明 |
|------|------|
| 認知健康訴求幾乎空白 | 葉黃素是腦中最豐富的類胡蘿蔔素（佔 66–77%），但僅 BioJOY 一家明確切入認知訴求，市場教育嚴重不足 |
| 兒童市場正在快速形成，台灣有先機 | 2025 年 Kemin 兒童臨床數據（+13% 認知、+14% MPOD、-6% 數位眼疲勞）推動新品上市，台灣已有 BalanStart、富宥家，但全球品牌仍稀少 |
| 孕婦 / 嬰幼兒市場幾乎未開發 | FloraGLO® 是唯一通過 FDA-GRAS 嬰兒配方認證的葉黃素品牌，但消費品市場只有 OcuGenics 與亞尼活力兩家針對孕婦推廣 |
| 女性美容市場全球幾乎空白 | 皮膚光保護（4倍）、含水量（+60%）、彈性（+20%）均有 RCT 證據，但全球僅 Glotrition 一家定位為美容產品，機會巨大 |
| 運動恢復訴求未被開發 | 葉黃素抗氧化機制適合高強度訓練後眼肌保護，僅 Bowmar Nutrition 切入，但其主訴求仍是「眼睛健康」而非「運動恢復」 |
| 日本市場高劑量空白持續 | 機能性表示食品法規下日本品牌普遍 10–16mg，20mg+ 高劑量市場幾乎空白，兒童專屬品牌亦只有 Rohto 一家 |

## 供應商資訊

**FloraGLO® 製造商**：Kemin Industries, Inc.（美國愛荷華州，1961年創立）

- 全球超過 2,600 名員工，26 個國家製造及服務設施
- 1996年推出世界首支品牌葉黃素 FloraGLO®，全球市場佔有率第一
- 臨床研究超過 100 篇（含 20 篇 RCT，見 `research/research_index.md`）
- 主要競品原料：Lutemax® 2020（OmniActive），含 Meso-zeaxanthin，Ma 2016 Meta-analysis 顯示含 MZ 配方 MPOD 效果更優（⚠️ FloraGLO® 不含 MZ）
