## 功效聲稱（逐條中日對照）

### 核心原料：Liposomal PureWay-C™

原料「Liposomal PureWay-C™」是以玉米來源維生素 C，搭配米糠來源脂肪酸、泰代（タイダイ）來源柑橘類黃酮（シトラスフラボノイド）所組成的脂質體維生素 C。透過申請中的特殊配方專利，相較於未進行脂質體化的維生素 C，具有更高的吸收率與更長的體內滯留時間。

**（原文：** 原材料の「Liposomal PureWay-C™」は、とうもろこし由来のビタミンCに、米ぬか由来の脂肪酸、タイダイ由来のシトラスフラボノイドを配合したリポソームビタミンC。特許申請中の特殊な配合により、リポソーム化していないビタミンCと比較して吸収率が高く、体内での滞留時間が長いのが特徴です。**）

備注：Liposomal PureWay-C™ 為 One Innovation Labs 公司商標。

### 脂質體作用機制

脂質體是由與人體細胞膜相同結構的磷脂質所構成的 50～150 nm 微膠囊，能將維生素 C 包覆於內部，抵抗消化液影響與氧化，緩慢溶解後精準輸送至目標部位。

**（原文：** リポソームとは、人の細胞膜と同じ構造のリン脂質でできた50～150nmのマイクロカプセル。内側にビタミンCを閉じ込めておくことで、消化液の影響や酸化からガードしながら少しずつ溶けていき、届けたい場所にしっかりと運ぶことが可能です。**）

### 體內滯留優勢

一般維生素 C 即使大量攝取，2～3 小時後便會排出體外。neutrial 的脂質體維生素 C 能讓維生素 C 緩慢且確實地被吸收。

**（原文：** ビタミンCは、摂取量が多くても2～3時間で体外に排出されてしまう成分ですが、ニュートリアルのリポソームビタミンCなら、ゆっくり着実に吸収させることが可能なのです。**）
