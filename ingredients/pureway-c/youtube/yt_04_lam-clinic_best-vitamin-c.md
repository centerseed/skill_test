## 影片摘要

本影片由 Dr. Carrie Lam 與 Dr. Michael Lam 共同介紹市面上各種維生素 C 的劑型與傳遞系統，包含水溶性抗壞血酸（Ascorbic Acid）、礦物質螯合型（如 Calcium Ascorbate、Magnesium Ascorbate）、脂溶性棕櫚酸抗壞血酯（Ascorbyl Palmitate）、脂質體（Liposomal）以及靜脈注射（IV）等形式。兩位醫師強調，各種形式各有優缺點，沒有單一「最佳」選擇——重點在於依個人身體狀況與使用時機，策略性地混合搭配不同傳遞系統，才能最大化維生素 C 的療效。對於一般健康人，普通超市產品即可；但若有特定健康需求，就必須在生化層面優化傳遞效率。影片最後討論了 IV 維生素 C 的使用場景與限制，認為在腎上腺疲勞的情境下，口服形式若使用得當已足夠，IV 通常並非必要。

## 重點整理（中文）

### [00:00] 開場：維生素 C 種類繁多，選擇困難

Dr. Carrie Lam 與 Dr. Michael Lam 開場指出，走進商店就會看到整排維生素 C 產品，令消費者不知從何選起。從大分類來看，維生素 C 的傳遞系統可分為四大類：水溶性（Water-soluble）、脂溶性（Fat-soluble）、脂質體（Liposomal），以及靜脈注射（IV）。本影片的目的便是逐一解析這些類型，讓消費者對傳遞方式有更清楚的認識。

### [00:34] 傳統水溶性：抗壞血酸（Ascorbic Acid）

歷史上，最純粹的維生素 C 形式稱為抗壞血酸（Ascorbic Acid），可製成粉末、膠囊或錠劑。在過去約 30 到 40 年間，這是主要被使用的形式，學術研究也大多以此為標的。

抗壞血酸的核心特性是快速吸收（fast absorption），但問題在於它同樣排泄快速（fast excretion），導致血中濃度像溜溜球（yoyo）一樣急升急降。對於健康且偶爾需要快速補充活力的人來說，這種特性是可以接受的；然而，若身體較虛弱或較為敏感，過度且頻繁地刺激系統反而可能造成耐受性下降，甚至崩潰（crash）等問題。正是這個原因，抗壞血酸在某些族群中逐漸失去吸引力。

### [01:54] 礦物質螯合型與脂溶性衍生物

為了改善抗壞血酸對腸胃的刺激性以及急升急降的問題，出現了兩種主要的改良方向。第一種是在抗壞血酸上附加礦物質，形成鈣抗壞血酸鹽（Calcium Ascorbate）或鎂抗壞血酸鹽（Magnesium Ascorbate），這樣做的目的是讓腸胃系統更容易接受。第二種則是附加脂肪化合物，以棕櫚（palm）脂肪酸為例，形成棕櫚酸抗壞血酯（Ascorbyl Palmitate）——這是一種脂肪與抗壞血酸的結合體，能降低其水溶性，從而使吸收曲線更為平緩，呈現出較慢的上升速度與較慢的下降速度，形成所謂「更溫和的傳遞曲線（more gentle delivery curve）」。這兩種形式仍屬口服劑型，主要仍透過腸胃道吸收。

### [02:44] 脂質體（Liposomal）技術

近 30 到 40 年間，科學界發展出以脂質體（Liposomes）奈米化技術封裝抗壞血酸的新型態。其運作原理是：脂質體將維生素 C 完整包覆，使其通過胃部（gastric mucosa）時不被釋放，而是等到進入小腸後才釋放，透過被動擴散（passive diffusion）的方式吸收。

這一差異的重要性在於能量消耗。若維生素 C 在胃部吸收，身體需要主動消耗能量，將抗壞血酸從管腔（lumen）運輸穿過黏膜（mucosa）；而在小腸透過被動擴散吸收，則完全不需消耗能量（no energy required），對身體的負擔因此大幅降低。Dr. Carrie Lam 接著做了一個確認性的總結：水溶性的維生素 C 被封裝在脂溶性的脂質體中，使其得以通過胃部、進入小腸，在那裡以不耗能的方式被吸收，之後脂質外殼消散，維生素 C 才真正被吸收進入體內。

### [03:52] 各劑型的策略性搭配

然而，各種形式都有其優缺點，因為有時候你需要快速傳遞——例如你需要即刻的能量、希望腎上腺獲得快速支持；有時候卻不希望它太快作用。影片的核心概念因此是：關鍵在於策略（The key is to have a strategy），而不是選定某一種或放棄另一種，而是如何針對自己的身體將它們混合搭配。

舉例來說，早晨起床時你可能需要快速能量，因此選擇水溶性抗壞血酸；夜間補充時若使用速效型，可能反而干擾睡眠，因此選擇慢釋型更為合適。要制定正確的維生素 C 組合，不只需要了解各種維生素 C 的特性，更需要了解自己的身體如何代謝維生素 C，並透過測試與協議（challenges and protocols）找出最適合的個人化組合。Dr. Michael Lam 強調，若未找到完美組合，許多補充都可能是完全的浪費，消費者可能自以為做對了，但身體卻沒有復原，然後便誤以為「這個東西沒效」——其實問題出在組合策略而非維生素 C 本身。有時需要高劑量，有時需要低劑量，有時甚至需要暫停補充。

### [05:08] 多種傳遞系統可同時使用

了解不同傳遞系統不只是有益的，對於有特定健康需求的人，同時使用多種傳遞系統更是理想的做法。若只是健康的人想補充少量維生素 C，去超市買任何一種都無妨；但若希望維生素 C 真正發揮健康支持的效果、協助身體復原，就必須在傳遞機制與生化層面上進行最大化與優化。

### [05:45] 靜脈注射維生素 C（IV Vitamin C）

IV 維生素 C 目前相當流行，劑量範圍從 10 克到 100 克不等，在一定時間內注射完畢。這不是可以居家自行操作的療法，必須在醫師診所進行，通常還會搭配維生素 B 注射，也就是所謂的 Myers cocktail，或是加入谷胱甘肽（Glutathione）。由於直接進入血流，IV 的作用非常快速，但也同樣快速消退。

IV 維生素 C 常被用於癌症支持（cancer support），但在腎上腺疲勞（Adrenal Fatigue）的情境下，兩位醫師的臨床觀察是：若口服形式使用得當，已經可以取得很好的效果，不需要一週三次綁在靜脈點滴架上。兩位醫師並未否定 IV 的價值——對某些人而言，IV 仍然非常重要——但整體而言，它通常並非必要的選擇。
