以下是 PureWay-C® 的系統性回顧白血球滯留數據比較（Calder et al., 2025）：

Calder 等人（2025）在 Nutrients（Q1，IF=5.08）的系統性回顧中，對不同維生素 C 形式的白血球維生素 C 濃度進行系統比較。白血球是最關鍵的免疫相關生物標誌物，因為白血球中維生素 C 濃度可達血漿的 10–100 倍，且與免疫細胞功能呈正相關。

關於 Ester-C® 的白血球滯留數據：三篇 RCT 顯示 Ester-C® 在白血球滯留方面具有一致優勢。Moyad 等人（2008，n=15）報告 Ester-C® 在 24 小時時白血球維生素 C 濃度為一般抗壞血酸的 1.3–1.7 倍（p<0.0001）。Mitmesser 等人（2016，n=30）確認 Ester-C® 在 24 小時白血球滯留顯著優於一般抗壞血酸（p=0.036）。Dickerson 等人（2024，n=93）進一步顯示 500 mg Ester-C® 可改善中性粒細胞吞噬作用及 NK 細胞活性。

PureWay-C® 的白血球數據缺口：Pancorbo 等人（2008）的人體試驗作為 PureWay-C® 的唯一臨床數據，僅測量血清維生素 C 濃度，未測量白血球維生素 C 濃度。Calder 等人在回顧中明確指出此為 PureWay-C® 證據的核心缺口，並要求「需要進一步研究，包括細胞或組織中的維生素 C 濃度」。

回顧整體結論：Ester-C® 在白血球滯留方面擁有最強的多研究一致性證據，直接關聯免疫功能；PureWay-C® 的血清吸收峰值優勢雖有數據支持，但目前無白血球層級數據，無法直接推論免疫功能優勢。兩種形式在免疫功能聲稱上的科學證據強度存在顯著差距，Calder 等人認定 Ester-C® 整體科學基礎目前較為強大。
