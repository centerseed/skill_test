以下是含 PureWay-C® 的市場產品資料（荃贏全美 allwealth，台灣）：

荃贏全美密碼有限公司出品的植萃維他命 C 膠囊採用美國 ONE INNOVATION 製造的 PureWay-C®，每顆含維生素 C 500 mg（總重 760 mg），30 顆 / 袋，立體鋁箔袋裝。PureWay-C® 以美國製程專利 US7297659B2 為技術背書，由 93% 純度非基改玉米發酵維生素 C、米糠脂質及柑橘生物類黃酮構成。製造商 ONE INNOVATION 持有 NSF GMP 認證工廠及 FDA 註冊，品牌榮獲 2026 世界品質評鑑大賞 2 特金 + 1 金。

品牌定位強調「植萃、93% 高純度、24H 長效釋放、100% 食物萃取」，訴求效力勝西印度櫻桃近 4 倍，引用 Med Sci Monit 2008 年文獻支持吸收優勢。衛福部核准功效宣稱包含：抗氧化、有助傷口癒合、增進結締組織及骨骼牙齒生長、促進膠原蛋白形成、維持皮膚正常功能、促進鐵吸收。

認證方面，原料持有 HALAL、Kosher（Orthodox Union）、GRAS 及 ISO 9001 四項認證，成品通過塑化劑、防腐劑、微生物及重金屬第三方安全檢驗，報告線上可查詢。通路以品牌官網為主。
