以下是含 PureWay-C® 的市場產品資料（Dr.Nutri，台灣）：

Dr.Nutri 高濃度緩釋型 PureWay-C 750 由台灣芯漾生醫股份有限公司製造，透過 mypoint 電商平台銷售，售價 NT$1,206（60 粒，玻璃罐裝）。每粒含 PureWay-C® 750 mg（較市場主流 500 mg 提升 50%），營養標示維生素 C 為 700 mg（PureWay-C® 複方中純維生素 C 含量約 93%，750 × 93% ≈ 697.5 mg），搭配柑橘生物類黃酮，採透明植物膠囊，無添加色素香料，台灣製造。

品牌以「高濃度、高質感玻璃罐」差異化，訴求植物脂質載體技術使吸收及保留率較傳統維生素 C 提高 233%，93% 高純度，溫和不刺激。衛福部核准功效宣稱包含：促進膠原蛋白形成並有助傷口癒合、維持細胞排列緊密性、增進結締組織及骨骼牙齒生長、促進鐵吸收、具抗氧化作用、維持牙齦與皮膚正常功能。

認證方面，PureWay-C® 原料持有多項國際認證；成品通過 SGS 多項第三方檢驗（重金屬、大腸桿菌群），提供 QR Code 即查報告。食品業者登錄字號 D-142931667-00000-9，投保產品責任險。
